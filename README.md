# game-about-animals
Mila's Forest Adventure, Unity motoru kullanılarak geliştirilmiş, 2D platform ve interaktif bulmaca/bilgi yarışması türlerini harmanlayan bir macera oyunudur. Oyuncular, ana karakter Mila'yı kontrol ederek ormanın derinliklerindeki gizemleri çözer ve hayvan dostlarıyla etkileşime girerek "Ormanın Kalbi"ni uyandırmaya çalışır.
